10. Settings & Configuration

Clicking the Settings ( ) button at the bottom of the sidebar opens the configuration modal, divided into three tabs.

1. Preferences​

  • Root Folder Location: This defines where Markd operates. If it is set to Other Bookmarks, Markd will completely ignore your Bookmarks Bar. You can safely change this location at any time.
  • Enable Compatibility Mode: For users on older laptops or vast bookmark collections, checking this box slows down internal loops and reduces animation complexity to prevent Chrome from freezing during massive operations.
  • Auto-Sync on Start: (Premium) Forces a non-destructive "Find & Refresh Plan" operation every time the extension is opened, ensuring you always see the freshest data.
  • Max HTML Size: A slider that controls how much webpage text Markd's scraper extracts before cutting it off. Lower sizes (100KB) process faster and use fewer API tokens. Higher sizes (2000KB) provide the AI with better context for complex sites.
  • Reset Core Settings: A panic button. If you break your URL patterns or corrupt an API connection, this button factory-resets the technical settings (Provider to Groq, default patterns, etc.) without deleting your actual folders or saved API keys.

2. AI Provider Configuration​

Markd allows you to hot-swap between multiple AI brains. The app intelligently hides the settings for inactive providers to keep the UI clean.

  • Cloud Providers (Groq, OpenAI, Google, Anthropic):
    • Paste your API Key and click Save Key. Markd will securely encrypt it locally and ping the provider's server.
    • If the key is valid, the Model Dropdown will populate with the latest available models (e.g., llama3-70b, gpt-4o, claude-3-5-sonnet).
    • API Key Reverts: If you type an invalid API key, or accidentally delete your key and hit Escape or click away, Markd will automatically revert the input box to the last known valid key. You never have to worry about accidentally breaking a working connection.
    • Anthropic Overrides: Because Anthropic API recently made it possible to download the model list, Markd provides a hardcoded list of the best Claude models in case this list download fails. You can click the Edit button next to the dropdown to assign a friendly custom name (e.g., "My Claude") or manually input a hyper-specific, brand-new model ID if Anthropic releases one to be able to use anthropic inserting manually the model in the list.
  • Local AI (Ollama):
    • Provides complete privacy by running the AI on your own hardware.
    • Ensure Ollama is running, input your local URL (default: http://localhost:11434), and click Connect.
    • Single Prompt Mode: Some local models fail to respond correctly to standard "Chat" API formatting. Checking this box forces Markd to use the older /api/generate endpoint, which often fixes connection timeouts on smaller models.

3. URL Patterns​

This tab allows you to configure the AI-Bypass rules described in Part 6.

  • Enable Smart URL Patterns: A master switch to turn the entire feature on or off.
  • Adding Patterns: The format must be exact. The left box takes a domain (e.g., youtube.com). The right box takes a specific category path (e.g., Entertainment/Videos).
  • Validation: Markd validates as you type. It will throw a red error if you include spaces in the domain, use an invalid format, or attempt to create a duplicate rule. You cannot save the list until all red errors are fixed.
  • Import / Export: You can export your ruleset to share with others. Importing a .json file will completely replace your current list with the new one.
