6. Bulk Operations & The FAB

Markd replaces tedious manual clicking with the Floating Action Bar (FAB), a powerful tool that slides up from the bottom of the screen whenever you select multiple items.
It always displays the total number and type of the selected items. There is also a "Clear" button to remove all selections and escape from the bulk selection mode; hitting the Esc key does the same.

1. Mutual Exclusivity​

Markd enforces strict separation between Folder operations and Bookmark operations.

  • If you check a folder's checkbox, all bookmark checkboxes are instantly disabled.
  • If you check a bookmark, all folder checkboxes are disabled.
  • A toast message will remind you: "Clear your folder selection to select bookmarks."

2. Smart Selection Filters​

The Controls Bar at the top contains checkboxes like Main Folders, Subfolders, and Empty Folders.

  • Strict Mode Locking: Clicking one of these boxes locks the UI into that mode. For example, if you click Empty Folders, Markd will instantly select every folder that has 0 bookmarks (A main folder considered empty if it does not contain any bookmarks or/and containes only empty subfolders). It will then physically disable the checkboxes for every folder that does contain bookmarks, preventing you from making a mistake.

3. Contextual Selection (Smart Subitem Selection Logic)​

The FAB is context-aware. If you manually check a subfolder inside the "Finance" category for example, the FAB will automatically generate a Smart Button that says: Select all in 'Finance'.

  • Clicking this will instantly check every subfolder inside that specific parent category.
  • If you click it again, the button intelligently flips to Deselect all in 'Finance'.
  • This selection method is universal for all selectable subitem.

4. Bulk Folder Operations​

When multiple folders are selected, the FAB adapts its buttons based on the type of folders you selected (determined by the Mode Locking in the Controls Bar):

  • Bulk Rename: Opens a powerful modal allowing you to apply Find & Replace, Add Prefix (e.g., [Archive]), or Add Suffix to dozens of folders simultaneously.
  • Bulk Delete: Permanently deletes the folders and their contents. Markd calculates the total bookmarks across all selected folders and warns you exactly how many items you are about to destroy.
  • Accept / Reject Proposals: If you used the Controls Bar to select "Flat Proposals", the FAB will show "Accept" (which runs the flattening operation on all of them) and "Reject" (which deletes the AI's suggestions and the corresponding deep folders from the browser too!).
  • Sort Within: If you select multiple physical Main Categories and at least one of the selected folders contains direct bookmarks, you can command the AI to sort bookmarks strictly inside those specific folders simultaneously.

5. Bulk Bookmark Operations & The "Move To..." Modal​

When multiple bookmarks are selected, the FAB allows you to Bulk Delete, Copy URLs, Move to Root (extracting them from their folders and moving all selected bookmarks to the active root folder), or Move to...

Clicking "Move to..." opens the Destination Modal:

  • This modal lists every valid folder in your workspace, cleanly styled to match their current status (Green for active, Blue for Plan, Red for Missing etc.).
  • System Folders (Root, Broken Links, Local Development) are pinned to the top of the actual branch.
  • The modal features a built-in search box. As you type, the list filters instantly, highlighting matches so you can route 50 bookmarks to a specific subfolder in seconds.
  • Safety Feature: The modal automatically disables and greys out the folder where the bookmarks currently reside, preventing you from moving them into themselves.
