What Is Markd?​

Markd is a professional, AI-powered Chrome extension that acts as a smart organizational layer directly on top of your native browser bookmarks. Unlike proprietary managers that lock your data away, everything Markd does — creating folders, moving links, building structure — happens inside Chrome's own local storage. Uninstall the extension at any time and your bookmarks remain perfectly intact.

Key Concepts at a Glance​

Active — physical folder in Chrome with bookmarks
Empty — folder exists but has 0 bookmarks
Plan — Blueprint-only, not yet in Chrome
Missing — deleted in Chrome, still in Blueprint
Conflict — duplicate name or flattening proposal
Protected — AI-locked, cannot be sorted or deleted
Local Dev — catches localhost & private network URLs

The Golden Rule​

Always Back Up First

Because Markd performs bulk physical operations on your Chrome bookmarks, always export a native Chrome HTML backup before running large AI sorts or structural changes. See Part 9, Section 4: The Chrome Backup Anomaly for the full backup guide.
